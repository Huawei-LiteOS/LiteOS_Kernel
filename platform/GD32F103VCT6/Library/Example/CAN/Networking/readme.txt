  @brief
    This demo is based on the GD32107C-EVAL board, it shows how to configure the CAN 
  peripheral to send and receive CAN frames in normal mode.The frames are sent and print
  transmit data by pressing User Key push button. When the frames are received, the receive
  data will be printed.
 
  @note
    When using CAN2, please check that your device is Connectivity line. 
    P2,P3 and P4 must be fitted.
    This example is tested with at least two GD32107C-EVAL board. The same program example is 
  loaded in all board and connect CAN_L and CAN_H pin of CAN1 or CAN2 to bus for sending 
  and receiving frames.

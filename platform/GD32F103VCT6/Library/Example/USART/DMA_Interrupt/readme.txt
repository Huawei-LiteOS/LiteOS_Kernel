  @brief
    This demo is based on the GD32107-EVAL board, it provides a basic communication
 between USART1 and USART2 using DMA and interrupts.USART1 use DMA transfer and RBNE
 flag receive ,USART2 use DMA transfer and interrupt receive.
  @note
     connect USART1_Rx(PA10) to USART2_Tx(PA2) 
     connect USART1_Tx(PA9)  to USART2_Rx(PA3)

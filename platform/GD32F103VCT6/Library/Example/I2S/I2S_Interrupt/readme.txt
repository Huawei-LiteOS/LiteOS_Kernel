  @brief
    This demo is based on the GD32107-EVAL board,The function of this demo is
  transfer data from I2S3 to I2S2.
 
  @note
    Connect I2S2 WS  PIN(PB12) TO I2S3 WS   PIN(PA4)
    Connect I2S2 CK  PIN(PB13) TO I2S3 CK   PIN(PC10)
    Connect I2S2 SD  PIN(PB15) TO I2S3 SD   PIN(PC12)

  @brief
    This demo is based on the GD32107C-EVAL board, it shows how to configure
  the CRC module. In this demo, 1 CRC code is calculated for a 32-bit data by 
  configuring various control bits in the CRC control register.

    If the test passes, the success_flag variable equals 0x1 and light the LED2.

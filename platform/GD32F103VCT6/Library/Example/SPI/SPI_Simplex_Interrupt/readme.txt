  @brief
    This demo is based on the GD32107-EVAL board,The function of this demo is
  transfer data from SPI1 to SPI3.SPI1 is set as master and SPI3 is slave.They 
  are work in simplex mode using Tx interrupt for master and Rx interrupt for slave.
 
  @note
    Connect SPI1 SCK  PIN(PA5) TO SPI3 SCK  PIN(PC10)
    Connect SPI1 MOSI PIN(PA7) TO SPI3 MISO PIN(PC11)

  @brief
    This demo is based on the GD32107-EVAL board, This demo provides a basic 
communication between USART1 (Synchronous mode) and SPI1. 
  @note
    Connect SPI1 SCK  PIN(PA5) TO USART1_CK  PIN(PA8)
    Connect SPI1 MISO PIN(PA6) TO USART1_RX  PIN(PA10)
    Connect SPI1 MOSI PIN(PA7) TO USART1_TX  PIN(PA9)
